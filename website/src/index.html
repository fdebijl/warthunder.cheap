About Warthunder.cheap

This website checks the War Thunder store every 12 hours and records all the prices for each item. If an item is on sale it will appear at the top of this page, showing the discount over the normal price. Clicking on 'View Details' will show you more information about the item, including a description, screenshots and price history graph. If you wish to support a creator and get a 3% discount for doing so, you can select a referal code in the header.

Further down the page you'll see all the items that are no longer available, but you can still view their details and price history. For any item, both current and archived, you can set alerts for availability or a discount.

If an alert triggers you'll receive an email with a link to the item on the War Thunder store. These alerts are one-time, so you'll only receive a single alert when the item goes on sale, after the alert email is sent we remove your email from the database. If you wish to receive the same alert again, you'll have to set it up again here.

Warthunder.cheap is fully open source, you can view the source code on Github.

FAQ

Frequently Anticipated Questions I guess

When is the next sale? The War Thunder store typically has sales around the following dates, but individual vehicles can go on sale outside of these events.

  • VE day/Victory Day/Victory in Europe day (~6th of May)
  • D-Day (~6th of June)
  • Summer Sale (Late June, early July)
  • War Thunder Anniversary (Late October)
  • Black Friday (Late November, on Black Friday)
  • Christmas/New Year's Sale (Late December)
Shoutout to /u/PureRushPwneD for compiling this list.

Can I see the prices in USD? As of 2025, USD and EUR prices in the War Thunder store are the same, so an item that's 60 euros will also be 60 dollars.

I changed the referal code but it still shows the previous one on the store Referal codes in the Gaijin Store are cached for an hour through cookies, clear your cookies and you should be able to select a new referal code.
