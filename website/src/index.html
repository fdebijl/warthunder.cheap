About Warthunder.cheap

This website checks the War Thunder store every hour and records all the prices for each item. If an item is on sale it will appear at the top of this page, showing the discount over the normal price. Clicking on 'View Details' will show you more information about the item, including a description, screenshots and price history graph. You can also enable alerts for discounts/availability in this details dialog.
If you wish to support a creator and get a 3% discount for doing so, you can select a referal code in the header. This code will automatically be applied to any outbound store links you click.

Further down the page you'll see all the items that are no longer available, but you can still view their details and price history, if available. For any item you can set alerts for availability if they're archived or a discount if they're currently available. The information on these items were retrieved from the Archive.org's Wayback Machine, so it's possible that items may miss descriptions, screenshots, pack contents or price history.

If an alert triggers you'll receive an email with a link to the item on the War Thunder store. These alerts are one-time, so you'll only receive a single alert when the item goes on sale, after the alert email is sent we remove your email from the database. If you wish to receive the same alert again, you'll have to set it up again here. You can also setup alerts for new items, to receive those you'll have to login under the 'Manage Alerts' dialog. Enter your email to receive a login link in your inbox, once logged in through that link you can manage your alerts for any item and enable/disable global alerts for new items. The new item alerts will trigger every time new items are added, unlike the discount and availability alerts.

Warthunder.cheap is fully open source, you can view the source code on Github.

FAQ

Frequently Anticipated Questions I guess

When is the next sale? The War Thunder store typically has sales around the following dates, but individual vehicles can go on sale outside of these events.

  • VE day/Victory Day/Victory in Europe day (~6th of May)
  • D-Day (~6th of June)
  • Summer Sale (Late June, early July)
  • War Thunder Anniversary (Late October)
  • Black Friday (Late November, on Black Friday)
  • Christmas/New Year's Sale (Late December)
Shoutout to /u/PureRushPwneD for compiling this list.

Does the 3% creator discount stack with normal discounts? No, it only applies to item that are not discounted, nor does it apply to golden eagle packs. In any event, you will still receive the decal if you don't already have it on your account.

Can I see the prices in USD? As of 2025, USD and EUR prices in the War Thunder store are the same, so an item that's 60 euros will also be 60 dollars.

I changed the referal code but it still shows the previous one on the store Referal codes in the Gaijin Store are cached for an hour through cookies, clear your cookies and you should be able to select a new referal code.

There's some days missing in the charts :( Warthunder.cheap is still in development, meaning the scraper isn't 100% reliable yet, so it might miss a day here and there.
